Une première approche de la programmation en JavaScript


Pour programmer en JavaScript qui est le language de votre navigateur web (Firefox, Chrome, Safari..), il faut d'abord créer une page web minimale puis on insère du code à l'intérieur des balises script.
Le body représente l'aspect physique du corps de la page (ce qu'on y voit : texte, boutons, images...) tandis que le script s'occupe du fonctionnement logique de ce qui se passe sur la page (interaction éventuelle avec l'utilisateur, mise en place d'un algorithme, restitution du résultat sur la page).
Pour cette première approche, on retiendra l'importance d'utiliser la console interne au navigateur, de commenter son code et de savoir chercher dans le Guide JavaScript.

1)  Réalisation d'une page html élémentaire (HyperText Markup (balise) Language)

</head>

//le code du script (un commentaire en JavaScript sur une ligne)

</script>

2)  Un premier programme: entrer une valeur dans un input, appeler une fonction en cliquant sur un bouton, récupérer une valeur

*********************************************** 
<html>
<head>
<meta charset="utf-8" />
<title> Ma première page</title>
</head>
<body>
Entrer un nombre : <input id="nombre"/><br>
<button onclick="myFunction()">Envoyez</button><br>
Le carré de ce nombre est : <span id="reponse"></span>
<script type="text/javascript">
function myFunction() {
var valeur = Number(document.getElementById("nombre").value);//on récupère la valeur identifiée par "nombre" en transformant la chaîne de caractères entrée en nombre
var carre=valeur*valeur;
console.log(carre);//la console web est le premier outil du programmeur 
document.getElementById("reponse").innerHTML = carre;//on affiche la valeur calculée dans l'espace identifié par "réponse"
}
</script>
</body>

5) À vous de jouer !

Écrire un programme qui demande un mot ou phrase (une chaîne de caractères ou string) et affiche la lettre du mot la plus présente dans celui-ci, c'est-à-dire la lettre de plus grande occurrence

Indications :
- pour chaque lettre du mot, on recherche le nombre d'occurrence de celle-ci dans le mot, réaliser pour cela deux boucles imbriquées utilisant des variables différentes ; la première boucle sert à parcourir toutes les lettres du mot, la deuxième sert à faire la comparaison entre la lettre du mot et les autres lettres. On introduira une variable pour le comptage des occurrences d'une lettre du mot, variable qui devra être réinitialisée à chaque tour de boucle.
- on réalise ensuite une suite de valeurs (array) qui à chaque tour de boucle est augmentée de l'occurrence de la lettre du mot
- on recherche le maximum de cette suite en utilisant simplement Math.max(...suite))
- on recherche la position dans la suite de ce maximum à l'aide de suite.indexOf() ; par exemple si suite=[1,5,3], suite.indexOf(5) vaut 1
- on recherche et on affiche la lettre du mot correspondant au maximum d'occurrences ainsi que le nombre d'occurrence correspondant

N.B : Amélioration, ne pas utiliser la fonction Mat.max(...suite) mais rechercher un algorithme permettant de trier puis de prendre le maximum en s'inspirant de la vidéo Lego Bubble Sort.
Indications :
- créer d'abord la fonction sort(liste) qui admet liste comme paramètre et qui retourne la liste ordonnée
- adapter cette fonction dans l'application précédente pour qu'elle remplace la fonction Mat.max(...suite).
